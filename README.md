# Soil-Type-Land-Management-Console

## Overview
The Soil Type & Land Management Console is a Java-based console application designed to manage agricultural land records efficiently. It helps in maintaining soil details, crop recommendations, fertilizer usage, irrigation methods, farmer records, and weather-related information in a structured manner.

This project demonstrates the application of Object-Oriented Programming concepts such as classes, objects, inheritance, encapsulation, and file handling using Java.

## Features
- Farmer record management
- Land record maintenance
- Soil type information management
- Crop recommendation based on soil type
- Fertilizer details management
- Irrigation method management
- Soil testing information
- Weather data tracking
- File-based data storage using `records.txt`

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- File Handling
- Console-Based Interface

## Modules
### Farmer Management
Stores and manages farmer-related information.

### Land Management
Maintains land details such as area, ownership, and usage.

### Soil Management
Stores soil type details and supports soil testing data.

### Crop Management
Provides crop-related information based on soil suitability.

### Fertilizer Management
Maintains fertilizer recommendations and usage details.

### Irrigation Management
Stores irrigation techniques and related information.

### Weather Data
Tracks weather information relevant to agricultural planning.

## How to Run
1. Clone the repository:
```bash
git clone https://github.com/your-username/Soil-Type-Land-Management-Console.git
```

2. Navigate to the project folder:
```bash
cd Soil-Type-Land-Management-Console
```

3. Compile all Java files:
```bash
javac *.java
```

4. Run the application:
```bash
java ManagementSystem
```
